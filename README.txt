Requirements:
	1.Java 6 or above
	2.Any JAVA IDE (preferable: ECLIPSE).
	3.Spring boot plugin ( preferable: SPRING TOOL SUITE 3 plugin in ECLIPSE).

Steps:
	1.Before opening the project in IDE, make sure you have installed Spring Boot framework. Load main class of the project and run( NOTE: main class name ends with "...Application.java")
	2.Test cases are attached as text file in each tool's package.
	3.Run the project and open links(testcases) in any web browser(there you will get JSON output) or open "index.html" in "..front" folder of each project.
